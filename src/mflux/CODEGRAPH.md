## Summary

mflux is a comprehensive MLX-based image and video generation framework for Apple Silicon, implementing multiple diffusion model architectures (FLUX, FLUX2/Klein, FIBO, QwenImage, ZImage, SeedVR2, etc.) with support for text-to-image, image-to-image, inpainting, controlnet, LoRA training/loading, depth estimation, upscaling, and vision-language model integration. The project handles the full pipeline from model weight downloading/quantization/loading through transformer-based denoising to VAE decoding and image saving, with Apple Silicon-specific optimizations. It also includes tooling for CLI argument parsing, release management, and automated testing.

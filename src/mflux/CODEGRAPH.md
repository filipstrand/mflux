## Summary

mflux is a comprehensive MLX-based image and video generation framework for Apple Silicon, implementing multiple diffusion model architectures (Flux, FIBO, QwenImage, ZImage, SeedVR2, Flux2Klein, etc.) with support for text-to-image, image-to-image, inpainting, depth-guided generation, ControlNet, LoRA training/inference, VAE encoding/decoding, and various multimodal pipelines. The project handles the full lifecycle from model weight downloading, quantization, and loading through inference scheduling and image saving, with callback systems for memory management and stepwise output. It targets Apple Silicon hardware via the MLX framework and includes tooling for release management, shell completion generation, and comprehensive test infrastructure.
